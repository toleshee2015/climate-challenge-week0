# 🌍 Ethiopia Climate Data Analysis Dashboard

## Project Description
An interactive Streamlit dashboard for analyzing Ethiopia's climate 
data. Users can explore, filter, visualize and export climate datasets 
without any coding knowledge.

## Technologies Used
- Python 3.14
- Streamlit
- Pandas
- Plotly
- NumPy / SciPy

## Project Structure
climate-challenge-week0/
├── app/main.py       # Streamlit dashboard
├── data/             # Climate dataset
├── notebooks/        # EDA notebooks
├── requirements.txt
└── README.md

## Installation
git clone https://github.com/toleshee2015/climate-challenge-week0
cd climate-challenge-week0
pip install -r requirements.txt
streamlit run app/main.py

## Core Features
1. Data loading and auto-cleaning
2. Interactive filtering by numeric columns
3. Chart builder (6 chart types)
4. Correlation heatmap
5. Statistical summary
6. CSV export
